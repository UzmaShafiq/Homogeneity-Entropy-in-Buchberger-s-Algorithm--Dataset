# Code and data for Homogeneity Entropy in Buchberger's Algorithm

- `buchberger_impl.py` — Buchberger's algorithm with four selection strategies
- `phc_parser.py` — PHCpack CSV loader
- `benchmark_random.py` — benchmark driver for random datasets
- `benchmark_phc.py` — benchmark driver for PHCpack
- `data/` — input CSVs (PHCpack and random corpora)

Tested with Python 3.7.16, SymPy 1.10.1.
